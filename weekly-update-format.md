# Weekly Update Format

Posted to [CHANNEL/MEDIUM]. [OWNER] reviews before posting.

## Rules
- CONCISE. No padding. No filler. No LLM tells (no hyphens as bullets).
- Task titles are the default. Only rephrase if the title is too vague for the audience.
- Issues section only appears when there's something genuinely blocking or unexpected. Omit if nothing qualifies.
- Owner must review and approve before posting.

## Template

```
[PROJECT] Weekly — [date]

Goal: [current milestone]

Done:
[completed tasks this week]

Active:
[in progress tasks]

Next:
[upcoming tasks]

Issues:
[only if something is genuinely blocking or unexpected — otherwise omit this section]
```
